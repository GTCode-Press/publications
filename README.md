# GTCode Publications Archive

Citable archive of original investigations, articles, research papers, and policy documents published on [GTCode.com](https://gtcode.com).

Each publication directory contains:
- **README.md** — full text of the publication
- **CITATION.cff** — machine-readable citation metadata ([Citation File Format](https://citation-file-format.github.io/))
- **cite.bib** — BibTeX entry for LaTeX, Zotero, Mendeley, and other reference managers

## Sections

| Section | Description |
|---------|-------------|
| [investigation/](investigation/) | Public-interest investigations |
| [articles/](articles/) | Long-form original research and analysis |
| [papers/](papers/) | Research papers |
| [guides/](guides/) | Technical guides and research roadmaps |
| [reports/](reports/) | Technical reports |
| [policies/](policies/) | Editorial standards, ethics, and AI policy |

## How to Cite

**Cite a specific publication:** Navigate to its directory and use the `cite.bib` or `CITATION.cff` file.

**Cite this archive as a whole:**

```bibtex
@misc{gtcodepress,
  author       = {Lono, Ekewaka},
  title        = {GTCode Publications Archive},
  year         = {2026},
  publisher    = {GTCode Press},
  url          = {https://github.com/GTCode-Press/publications}
}
```

## Immutable References

Every file in this repository is version-controlled. To cite a specific version of any document, use a commit-level permalink:

```
https://github.com/GTCode-Press/publications/blob/<commit-sha>/investigation/the-federal-layer/README.md
```

Navigate to any file on GitHub, press **y** to convert the URL to a permanent commit-level link.
